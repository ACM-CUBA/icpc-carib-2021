There are $n$ points in a plane. Each point will be removed with probability $0.5$ independently. Determine the expected value of the area of the \textit{convex hull} of the remaining points.\\

\inputdesc{ The first line contains an integer $n$ $(1\le n\le 2000)$.\\

    The next $n$ lines contain integer pairs $x_i$, $y_i$ $(-10^9\le x_i, y_i\le 10^9)$, the coordinates of the $i$\text{-th} point. No three points are collinear.
}
